--- DESCRIPTION DU PROJET ---
EduPath est une application web de collecte et d'analyse descriptive des données. 
Elle vise à optimiser la réussite des étudiants en analysant les corrélations 
entre leurs habitudes (étude, sommeil, assiduité) et leurs performances.

--- CRITÈRES DU PROGRAMME INF 232 EC2 RESPECTÉS ---
L'application intègre les modules suivants du cours :
1. Collecte de données en ligne (via formulaire Flask).
2. Régression linéaire multiple (Prédiction de la note finale).
3. Réduction de dimensionnalité (PCA - Visualisation des axes d'influence).
4. Classification supervisée (Évaluation du niveau de risque).
5. Classification non-supervisée (Clustering K-Means pour définir les profils étudiants).

--- QUALITÉS DE L'APPLICATION ---
- Créativité : Secteur de l'éducation (EDUTECH) adapté au contexte de l'UY1.
- Robustesse : Gestion des erreurs de saisie et normalisation des données.
- Efficacité : Modèles de Machine Learning légers et rapides.

--- ARCHITECTURE TECHNIQUE ---
- Langage : Python 3.10
- Framework Web : Flask
- Bibliothèques Data Science : Scikit-learn, Pandas, NumPy
- Visualisation : Plotly (Graphiques interactifs 3D)
- Déploiement : Render (PaaS)
